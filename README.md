# MPF462972

Este repositorio contiene los archivos de la causa MPF462972 reorganizados y ordenados por fecha para poder documentarlos mejor en [Justicia por Margarita](https:///kosciuk.com.ar/margarita/index.html)

El [repositorio original](https://gitlab.com/kosciuk/mpf00462972) se encuentra en GitLab y ahi seguirá sin modificaciones.

## Modificaciones respecto al original
